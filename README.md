# arnaldosepulveda.com

Personal site for Arnaldo Sepulveda. Single static HTML page deployed via Cloudflare Pages.

## Purpose

Search-result destination and entity-consolidation target. Person schema (JSON-LD) links this site to my LinkedIn, GitHub, and Keystone AI project.

## Stack

Plain HTML and CSS. No framework, no build step.

## Deployment

Cloudflare Pages, auto-deploy from `main`.

## Edits

Edit `index.html` directly. Push to `main`. Cloudflare Pages redeploys in under a minute.
